# RadarPose-HybridNet
Gemini said This repository features a state-of-the-art Hybrid End-to-End Pipeline for Human Activity Recognition (HAR) using sparse mmWave radar. By unifying a Point Transformer for 3D pose and a CTR-GCN for temporal classification, it achieves 95.79% accuracy on 23 curated actions. Includes differentiable resampling and task-aware refinement.
